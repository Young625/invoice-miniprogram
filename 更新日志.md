# 贴票侠更新日志

## 版本 2.0.5 (2026年3月12日)

### 新增功能
1. 新增邮箱绑定校验
2. 报销包Excel新增OA单号字段

### 功能优化
1. 优化发票识别，包括发票类型识别和发票内容识别
   - 提升增值税专用发票、普通发票、电子发票等类型识别准确率
   - 优化发票号码、日期、金额等关键信息提取
   - 增强异常发票处理能力

2. 优化同步重复发票的情况，并新增友好提示
   - 避免重复报销，提升数据准确性

3. 优化生成报销包Excel内容
   - 按发票项目名称自动排序
   - 优化报销单据格式和布局

4. 优化小程序界面
   - 改进发票列表页面交互
   - 增强报销页面功能
   - 优化首页、设置页、详情页展示

5. 优化邮件服务
   - 提升邮件通知可靠性

6. 优化后端服务
   - 提升报销服务性能和稳定性
   - 优化数据库连接和查询
   - 改进API接口
