## 🔥 核心功能：邮箱实时监控

### 工作原理

1. **用户配置邮箱**
   - 在小程序"设置"页面配置邮箱信息
   - 支持 163、QQ、Gmail 等所有 IMAP 邮箱
   - 配置后立即生效

2. **自动监控（每 5 分钟）**
   - 后端定时任务自动运行
   - 检查所有配置了邮箱的用户
   - 提取未读邮件中的发票
   - **不会漏掉任何一张发票**

3. **智能提取**
   - 支持 PDF 附件
   - 支持邮件正文中的链接
   - 自动识别发票类型
   - 自动解析发票信息

4. **实时通知**
   - 新发票自动入库
   - 小程序首页实时更新
   - 统计数据自动刷新

## 🚀 快速启动

## ⚠️ 重要说明

### 邮箱授权码获取

**163 邮箱**：
1. 登录 163 邮箱网页版
2. 设置 → POP3/SMTP/IMAP
3. 开启 IMAP 服务
4. 获取授权码（不是邮箱密码！）

**QQ 邮箱**：
1. 登录 QQ 邮箱网页版
2. 设置 → 账户
3. 开启 IMAP 服务
4. 生成授权码

### 不会漏掉发票的保证

1. **定时任务可靠性**
   - APScheduler 是成熟的 Python 定时任务库
   - 任务失败会自动重试
   - 详细的日志记录

2. **邮件处理机制**
   - 只处理未读邮件
   - 处理后不标记为已读（避免误操作）
   - 数据库去重保证不重复

3. **错误恢复**
   - 单个邮件失败不影响其他邮件
   - 单个用户失败不影响其他用户
   - 下次轮询会重新处理

4. **监控和日志**
   - 所有操作都有日志
   - 可以查看日志确认运行状态
   - 出错会记录详细错误信息
